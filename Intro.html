Academic Projects

1. Title: Digital Image Forgery Detection using ML [GRIET, Hyderabad]
Jul 2022 - Apr 2023

•The project aims to detect morphed images. • The Machine takes an original reference image and a modified image as inputs and illustrates the morphed region if any morphing is done on that image. • We used Python as our programming language, and we used algorithms like Support Vector Machine and Random Forest. • It can be used to detect fake certificates, fake images on social media, and banking.

2. Title: Adenocarcinoma Detection [GRIET, Hyderabad]
Jan 2022 - Jun 2022

• The main objective of the project is to detect the region of tumor cells present in the lungs of a human being. • The machine takes a CT scan as an input and gives the circled region of tumor cells present in the lungs as output with more than 97 percent accuracy. • We used CNN as our primary algorithm, python as a programming language, and certain web technologies to build the user interface. The Machine is trained with numerous CT scans of human lungs as the dataset.
